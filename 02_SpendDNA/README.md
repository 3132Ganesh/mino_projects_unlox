# Spend DNA
### wallet's year-end story

Built for The Unlox Academy — Minor Project 2.
Parses a bank statement CSV, categorizes transactions by vendor, and produces
spend analytics by category (Food Delivery, Ecommerce, Transport, Subscriptions, etc.)

## Files
- `spend_dna.ipynb` — main analysis notebook
- `data/transactions_june.csv` — sample transaction dataset
- `DS_June_Minor_Project_2_Brief.pdf` — assignment brief
